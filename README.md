# ⚡ 极客工具箱 · Geek Toolbox

## 🔗 在线访问

## 🧰 工具列表

| 工具 | 功能 |
|------|------|
| JSON 格式化 | 格式化/校验/压缩 JSON |
| 文本对比 | 两栏对比文本差异，高亮显示增删 |
| Base64 编解码 | 编码/解码 Base64 |
| 字数统计 | 实时统计中文字数、英文单词、行数、标点等 |
| URL 编解码 | 编码/解码 URL 字符串 |
| 时间戳转换 | 时间 ↔ 时间戳互转，实时当前时间 |

## 🛠 技术栈
